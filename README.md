## 关于PI数据库

PI数据库是一款实时数据库，类似于时序数据库，广泛应用于工业领域，尤其是在电厂等场景中。它能够实时记录和查询点位的当前快照，同时也支持查询特定时间区间内的数据，为工业数据管理提供了强大的支持。
